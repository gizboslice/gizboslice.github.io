Motion Sensor Light

This project is an introduction to basic circuitry and Python programming with the Raspberry Pi. The ultimate result will be a light that is activated when the PIR sensor recognizes high activity.

Through these modules you will...

  • Create a circuit that gets a light to turn on
  • Successfully write a complete python program
  • Set a light to blink every second
  • Connect a motion sensor to your Pi
  • Get a light to blink when the motion sensor is activated

Things you need...

  • Raspbery Pi with basic peripherals (mouse, keyboard, monitor, etc.)
  • LED light and resistor
  • Male-to-Male and Female-to-Female jumper wires
  • PIR Motion Sensor

  1. Connect the LED light to pin 3 of the Raspberry Pi GPIO.

  2. Open a text editor, write the following code, and save it as a Python File.

    3. import RPi.GPIO as GPIO
import time
GPIO.setwarnings(False)
GPIO.setmode(GPIO.BOARD)
GPIO.setup(3,GPIO.OUT)       #Define pin 3 as an output pin

while True:
        GPIO.output(3,1)     #Outputs digital HIGH signal (5v) on pin 3 
        time.sleep(1)        #Time delay of 1 second

        GPIO.output(3,0)     #Outputs digital LOW signal (0v) on pin 3 
        time.sleep(1)        #Time delay of 1 second

  3. Run the program on your Raspberry Pi to test your work.

  4. Attach the PIR Motion Sensor to your Raspberry Pi like the image indicates

  5. Open a text editor, write the following code, and save it as a Python File.

    6. import RPi.GPIO as GPIO
import time
GPIO.setwarnings(False)
GPIO.setmode(GPIO.BOARD)
GPIO.setup(11,GPIO.IN)                   #Read output from PIR motion sensor
GPIO.setup(3,GPIO.OUT)                   #Define pin 3 as an output pin

while True:
    i = GPIO.input(11)       
    if i == 0:                           #When output from motion sensor is LOW
        print "No intruders", i
        GPIO.output(3,0)                 #Turn light off
        time.sleep(1)
    elif i == i:                         #when output from motion sensor is HIGH
        print "Intruder detected", i     
        GPIO.output(3,1)                 #turn light on
        time.sleep(1)

  6. Run the program on your Raspberry Pi to test your work.
