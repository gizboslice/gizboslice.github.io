SAY HI TO PI

Raspberry Pi is a low cost, open-source, Unix based, mini-computer that gives anyone not only an opportunity learn how to program in languages like Scratch and Python, but it also allows the ability experiment in designing and creating powerful electronics that can interact with the outside world.

The goal of these modules is to provide a place where any type of person of any age can learn how to build brand new electronic devices.

With the correct recources, anyone can become a thought leader in this industry.
