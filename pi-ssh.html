Let's set up an SSH Connection

An SSH, or a Secure Shell, is an encrypted network protocol to allow remote login and other network services to operate securely over an unsecured network.

Here are the steps to set up an SSH on a Mac.

First enable SSH in your Pi's settings

  • Open your Pi's terminal, and open the software configuration tool by typing the following commmand:
    • sudo raspi-config
  • Open advanced options, select "SSH", and a message should pop up asking if you would like the SSH server enable or disabled. Select Enable.

Next make the connection on your device

  • Open your Pi's terminal, and open the software configuration tool by typing the following commmand:
    • hostname -I
  • Open the terminal on your Mac and write the following command, but replace "IP" with the IP address of the Raspberry Pi.
    • ssh pi@IP
  • When the connection works, you'll see a security authenticity warning. type "yes" to continue.
  • You will then be prompted for the pi login. On raspbian, the default password is "raspberry".

You are now connected to your pi remotely
